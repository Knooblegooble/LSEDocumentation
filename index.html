INTRODUCTION

The Live Stream Extension (LSE) is a fully modular, upgrade-able, Blueprint-able solution for all your live streaming needs!
Basically... a networking framework with fully featured extensions for Twitch & YouTube Live.


Supercharge your Unreal projects with modular HTTP, WebSocket, and OAuth2 pipelines—plus real-time Twitch & YouTube interaction, auto-emote rendering, chat-triggered gameplay, and live stream playback on any surface.

No external SDKs, no C++ or networking knowledge required—just clean, event-driven Blueprint logic with examples for Twitch and YouTube Live.

-- Highlights --

  -OAuth2 (Twitch, YouTube, custom providers)
  -HTTP/JSON with automatic Struct ↔ JSON conversion
  -WebSocket (direct & handshake)
  -Twitch & YouTube integration (auth, chat, emotes, video data)
  -RichText emote display via Slate with auto download support
  -.m3u8 extraction from stream links (Custom youtube-dl.exe interface provided)
  -Live playback on any material or surface
  -Real-time chat triggers → Blueprint events
  -AES-encrypted key storage (.ini, runtime + editor)
  -Utility functions: Regex, Enum tools, Base64 ⇆ Texture
  -Full Blueprint interface support for structured events

Perfect for:

  -AI chatbot interactions with live viewers
  -Stream-controlled gameplay
  -Secure live dashboards
  -In-engine video rendering
  -Spectator overlays
  -JSON-driven UI systems
  -YouTube Live and Twitch chat in a single feed
