LSE's Networking Toolkit Key Storage system provides a secure and convenient way to store sensitive API keys or credentials inside your Unreal project.
 
  ‚Ä¢ Keys are encrypted automatically and saved to Saved/Config/WindowsEditor/Game.ini. ('Set as default' copies these to DefaultGame.ini)

  ‚Ä¢ Use the Set Key In Storage and Get Key From Storage blueprint nodes to manage keys at runtime.

  ‚Ä¢ To include default keys in packaged builds, hit "Set As Default" after editing them in the settings panel.

  ‚Ä¢ Encryption uses AES with Base64 encoding.
